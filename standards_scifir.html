Standards of Scifir

Names of genes

Here is explained the nomenclature of genes of Scifir. Genes are named with a stantard name that always is related to its effect. For example, the name of the gene that causes the eyes to be brown following this standard is "brown-eyes".

Names of molecules

Here is explained the convention of molecule names of Scifir. This convention allows to use only the IUPAC nomenclature and, optionally, the common name of the molecule, if there's a common name defined for that molecule. If there isn't a common name, only the IUPAC nomenclature can be used.

File formats

Depending of the type of data to store, the following file formats should be preferred to be used prioritarily to other file formats of the syme kind of data:

  • Complete genome, chromosomes, genes and dna strands: Use the .dna file format.
  • Animals, plants and fungus: Use the .life file format for the life form. For the history of the life form use .hlf inside the .life file, and the .dna file format for its genetic information inside that same .life file. If that life form has been created using the Bioeditor, the .lml file must be stored, it should't be losed.
  • 3D representations: A 3d representation can be a representation of a life form or an object in real life, or a digital 3d graphic. 3d representations are stored using the L3D file format.
  • Molecules: Use a chemistry file format for molecules created in Scifir, currently not published.
  • Substances: Use a chemistry file format for substances created in Scifir, currently not published.
  • New words: Use the .cwd file format.

If there's no other choice than to use another file format, just use it.

Units

The scalar units of measurement are written following the ISO of the units of measurement. According to that ISO, the magnitude of the units if written followed by an empty space and, then, the dimensions of the units of measurement, each of which can be preceded by a prefix. Also, the dimensions are multiplied with the character *, and divided with the character /.

Mathematical objects

Vectors

Vectors, in 3D, are written with their scalar unit followed by the angle theta and then the angle phi of the spherical coordinates. An example of such case is "3 N 20° 35°".

Coordinates

Geographical coordinates

Here are explained the geographical coordinates of Scifir. This geographical coordinates of Scifir extend the geographical coordinates of the Earth to be used with any astronomical object in the universe.

Astronomical coordinates

The astronomical coordinates used vary related to the astronomical object where the coordinates to representes are located. Depending on that astronomical object the astronomical coordinates to use are:

  • Planet: The equatorial coordinates adapted to the shape of the planet are used, they are identical to the equatorial coordinates of the Earth but use the radius of the new planet.
  • Orbit of the planet: The equatorial coordinates are used, but with a negative radius, because the point is outside the planet.
  • Moon: The equatorial coordinates are used adapted to the shape of the moon.
  • Orbit of the moon: The equatorial coordinates are used adapted to the moon, but with a negative radius, because the point is outside the moon.
  • Asteroid: The equatorial coordinates are used adapted to the shape of the asteroid.
  • Orbit of the asteroid: The equatorial coordinates are used adapted to the asteroid, but with a negative radius, because the point is outside the asteroid.
  • Star system: The ecliptic coordinate system is used with the center being the center of the star of that star system.
  • Galaxy: The galactic coordinates are changed, instead of having the origin in the sun they have the origin in the galactic center.
