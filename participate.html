You can participate in Scifir!

Scifir allows open participation, which means that any scientist can participate in it. To do so, you can do any of the following actions:

  • Send issues you find in our code repositories inside GitHub by using the GitHub Issues of the corresponding repository.
  • You can join any open source project. If you want to do so, send an email to scifir.foundation@gmail.com
  • If you want to publish any paper, software or document through Scifir, send us an email to, possibly it can be accepted.
